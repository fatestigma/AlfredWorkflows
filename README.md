## TV Show Helper
update at Jun 28th, 2014

Make watching tv shows more easier.

**使用方法:**

>* `watch {show}` 快速查看{show}的当前进度  
* `watch undo` 撤销上一次修改，包括init、finish  
* `watch all` 查看当前所有片的进度  
* `finish {show}` 看完，从表中删除  
* `watch init {eps} {show}` 直接修改{show}的当前进度为{eps}
